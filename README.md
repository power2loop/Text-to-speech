# Text-to-speech

 🗣️ Text-to-Speech Converter
 
A simple and interactive Text-to-Speech web application that converts written text into spoken words using the Web Speech API.

---
✨ Features

🔊 Real-time voice output from user input text

🎤 Multiple voice options (if supported by browser)

⚡ Lightweight and fast — no external libraries required

---

🛠️ Technologies Used

HTML5 – Structure of the app

CSS3 – Styling and layout

JavaScript  – Core logic and Web Speech API integration

🚀 How to Use

Type or paste your text into the input box.

Select your preferred voice, rate, and pitch.

Click the Speak button.

The browser will read the text aloud.
